# TypeScript SDK

Open source library to interact with any EVM-compatible blockchain.

<Callout>
V5 is the latest version of the thirdweb SDK. If you are looking for documentation for previous versions, please visit the [V4 documentation](/typescript/v4).
</Callout>

### Features

- **React & React-Native Support:** Supported React & React Native hooks with UI components to use in your application.
- **Extension-Driven Framework:** Extensions automatically detects and enables functionality based on common standards such as ERC-20, ERC-721, ERC-1155, and more.
- **Any EVM Compatibility:** Operate with any EVM-compatible network without specifying ABIs, RPC URLs, or IPFS gateways.
- **High-Level Abstractions:** Simplifies complex contract interactions into easy-to-use APIs.
- **Flexibility and No Vendor Lock-in:** Modular and composable enabling customizations and replacements of the default settings and tools.
- **Interoperable:** Works side-by-side with other popular libraries and frameworks such as ethers v5, ethers v6, and viem. 

## Installation

```bash
npm install thirdweb@beta
```
